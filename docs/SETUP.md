# Setup inicial

## Requisitos

- Git
- Cuenta de GitHub
- (Opcional) Node.js o Python para el motor de generacion

## Primeros pasos

1. Clona el repo.
2. Copia `.env.example` a `.env`.
3. Rellena tu API key.
4. Mete tus pruebas en `input/`.

## Estructura de input recomendada

- `input/video.mp4`
- `input/referencia.png`
- `input/prompt.txt`
